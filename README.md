## 💬 AI 对话助手

### ChatGPT
- **官网**: https://chat.openai.com
- **开发**: OpenAI
- **定价**: 免费 + Plus ($20/月)
- **特点**: 通用对话最强，支持 GPT-4，多模态
- **适合**: 写作、编程、分析、学习

### Claude
- **官网**: https://claude.ai
- **开发**: Anthropic
- **定价**: 免费 + Pro ($20/月)
- **特点**: 长上下文（200K），安全对话，分析能力强
- **适合**: 长文分析、代码审查、内容创作

### Kimi (月之暗面)
- **官网**: https://kimi.moonshot.cn
- **开发**: 月之暗面
- **定价**: 免费
- **特点**: 支持20万字上下文，国产优秀
- **适合**: 长文阅读、资料整理、深度研究

### 豆包
- **官网**: https://www.doubao.com
- **开发**: 字节跳动
- **定价**: 免费
- **特点**: 国内访问流畅，语音对话强
- **适合**: 日常对话、写作辅助

### 秘塔AI
- **官网**: https://metaso.cn
- **开发**: 秘塔科技
- **定价**: 免费
- **特点**: AI 搜索引擎，有引用来源
- **适合**: 信息检索、学术研究
